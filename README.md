# projeto-musica-teste
# 🎵 Projeto Música

Este é um projeto simples criado para testar o uso do Git e do GitHub.

## 📌 Objetivo

Praticar os comandos básicos de versionamento de código e publicação de um projeto.

Comandos utilizados:

* git add
* git commit
* git push

## 🎧 Tema do Projeto

O tema deste projeto é música. A ideia é criar um pequeno espaço para registrar artistas, músicas favoritas ou ideias relacionadas ao universo musical.

## 🛠 Tecnologias

* Git
* GitHub

## 📂 Sobre o projeto

Este repositório foi criado como um projeto de teste para aprender como criar e publicar projetos no GitHub.
